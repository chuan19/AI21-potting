# AI21-potting
基于YOLOv5智能盆栽
预测荔枝蒂蛀虫成虫羽化高峰期有许多预测方法，但都存在许多缺陷，这些方法的采样步骤复杂，实验时间周期长，环境影响因素大，得到准确数据需要大量采样，还会存在预测时间的滞后。为了解决上述问题，采用人工智能计算机视觉技术准确地预测荔枝蒂蛀虫成虫羽化高峰期，这样可以节省人力物力并提高荔枝的产量和品质，取到更好的预期效果。通过Yolov5算法实现荔枝蛾检测识别；通过树莓派实现远程控制和图像采集处理，获取蛾虫的数量和种类并上传云端。通过web实现显示蛾虫具体信息。
重点解决问题：
1、用Yolov5算法对图像进行识别；
2、对蛾虫数量种类进行统计，上传云端并在页面显示；
3、模块之间的联动
4、产品的连接与组装
